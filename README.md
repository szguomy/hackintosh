## Host installed
* HeliPort for wifi
* HotkeyDaemon for hotkey and osd display

## Function Key


| Key  |  Function |
|---|---|
|  F2 |  screenLock |
|  Fn+F2 |  sleep |
|  Fn+F3~F12| functions as keyboard show|
|  DClick tpad l-t corner| function as Fn+F5|
